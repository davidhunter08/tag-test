When to use this component

Use the tag component to indicate the status of something, such as an item on a task list.

Using multiple tags

You can use multiple tags for things that have multiple states. For example, items in a task list.

Tags should help people to organise things into useful categories: for example, things they’ve done and things they still need to do. The more categories you add, the harder it is for users to remember them. So start with the fewest number of tags you think might work, then add more if your user research shows there’s a need for them.

Using coloured tags

You can use colour to help distinguish between different tags - or to help draw the user’s attention to a tag if it’s especially important. For example, it probably makes sense to use ´nhsuk-tag–red´ for a tag that reads ‘Urgent’.

Tags that have the same wording should use the same colour. Do not use colour alone to convey information because it’s not accessible.

Accessibility

All contrasts between text and background meet level AAA of WCAG 2.0. Read more about accessibility and colour.

Research on this component

GOV.UK

The Department for Education use the coloured tags in:

  • apply for teacher training (used by citizens)
  • manage teacher training applications (used by teacher training providers)

The Apply for teacher training services at Department for Education used the Tag component extensively. But the colours (white text on colour background) were found to have poor contrast in an accessibility DAC audit. They solved this by using tints.
